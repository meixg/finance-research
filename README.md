## 📊 研究项目

### [多起点定投策略对比分析](./projects/dca-multi-start-comparison/README.md)

> **问题**：从不同时间起点开始定投，收益差距有多大？不同标的和货币组表现如何？
>
> **回测区间**：2010-01-01 / 2015-01-01 / 2020-01-01 → 2026-06-16
>
> **策略**：每周二定投，CNY 1000元 / USD 150美元 / HKD 1167港币，费率 0.1%

**覆盖标的：**
- 🇨🇳 **人民币组**（1000元/周）：沪深300、创业板50、中概互联、黄金ETF、纳斯达克100
- 🇺🇸 **美元组**（150美元/周）：标普500(SPY)、全球除美(VXUS)
- 🇭🇰 **港股组**（1167港币/周）：恒生指数(2800.HK)

**核心结论：**

| 排名 | 标的 | 起点 | 定投收益率 | 年化(XIRR) |
|------|------|:---:|:---------:|:----------:|
| 🥇 | 纳斯达克100 | 2010 | **+304.68%** | +16.45% |
| 🥇 | 标普500 | 2010 | **+279.75%** | +14.72% |
| 🥇 | 纳斯达克100 | 2015 | **+275.82%** | +21.65% |
| 🥇 | 黄金ETF | 2010 | **+180.85%** | +12.41% |
| 🥇 | 纳斯达克100 | 2020 | +112.80% | **+23.46%** 🏆 |
| 📉 | 中概互联 | 2020 | **-16.32%** | -5.55% |

- 纳斯达克100 总收益和年化均表现最佳；中概互联是唯一亏损标的
- 定投在 62% 的组中跑输一次性投入，但降低了择时压力和波动风险
- 三个起点均有 7/8 标的正收益，黄金和纳斯达克100在所有起点均表现优异
